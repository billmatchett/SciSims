Air Flow Around Objects Simulator

Explore how object shape, size, angle and flow speed affect the way air particles move around an obstacle.

Learning Intention

Students will understand that the shape and angle of an object can change how air flows around it.

Success Criteria

Students can compare flow patterns around different shapes and explain how shape, size and angle affect particle paths, visible wake patterns and drag.

Air flows from left to right. The shaded region is a simplified wake guide showing where disturbed flow may occur.

Student challenge

  1. Start with the circle. Describe how the particles move around it.
  2. Change to the square. Look carefully near the corners and behind the object.
  3. Rotate the triangle / wedge and streamlined oval. Explain how the angle changes the flow.
  4. Choose the shape that disturbs the flow least and justify your answer using evidence from the model.

Model note

This is a visual model. It uses a simple velocity field to show particles being diverted around objects. It is not a full fluid dynamics, viscosity or turbulence simulation.

For the triangle / wedge and streamlined oval, rotation also adds a simple angle-of-attack effect so students can clearly observe how changing angle changes the disturbed flow pattern.

The visible disruption score is based on this model’s settings. It is not a real drag coefficient and should only be used to compare patterns within this simulator.
