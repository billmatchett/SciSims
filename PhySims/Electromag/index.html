Electromagnet Simulator

Investigate how current, coil turns and core material affect the strength and polarity of an electromagnet.

Physics Electromagnetism Fields Circuits

Learning Intention

Students will understand that an electric current in a coil can produce a magnetic field and that an iron core can make the electromagnet stronger.

Success Criteria

Students can explain how current, number of turns and core material affect electromagnet strength, and predict how reversing the current changes the magnetic poles.

Model note

This sim uses a simplified classroom model. Relative field strength increases with current, number of turns and core effect, but the values are not exact tesla measurements. Heat risk is estimated from current and coil turns, not from a measured resistance.

Student challenge

  • Keep the current fixed and change the number of turns. Describe the pattern.
  • Keep the number of turns fixed and change the current. Describe the pattern.
  • Compare no core, soft iron, steel and ferrite. Which creates the strongest electromagnet in this model?
  • Reverse the current and explain what happens to the north and south poles.
  • Find a strong electromagnet setting with only a moderate heat risk.

Teacher notes

This sim links magnetism and circuits by showing that a current-carrying coil behaves like a magnet. A soft iron core strengthens the electromagnet because it becomes magnetised while current flows, but it does not behave like a permanent magnet once the current is removed. It is designed for qualitative and semi-quantitative comparisons rather than precise electromagnetic calculations.

  • Good prompt: What evidence shows that increasing current increases the magnetic field?
  • Good prompt: Why does adding a soft iron core make the electromagnet stronger?
  • Extension: Ask students to plan a fair test for electromagnet strength using paperclips.
