Drag-and-Drop Punnett Square

Use the gene tiles to build the offspring genotypes. Each offspring receives one allele from each parent.

Set up the challenge

Gene tiles

How to use:
Drag gene tiles into the empty spaces in each offspring box. You can also click a tile, then click a space. Each offspring must have two alleles.
Teaching point:
The letters in each box can be in either order, but the dominant allele is usually written first, such as Yy instead of yY.
