Reaction Time Challenge

A classroom reaction-time simulator for comparing repeated trials. Students respond only when their lane changes to GO!. False starts are recorded, and valid trials are used to calculate summary statistics.

Enter student names, then lock names or press Start.
Keyboard shortcuts: Space start/respond, R reset. This sim runs sequentially so only the active student can record a response.
